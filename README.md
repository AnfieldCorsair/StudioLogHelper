# AI Studio Log Parser

Десктоп-приложение + CLI для парсинга логов чатов **Google AI Studio**
(JSON-файлы с Google Drive, в том числе скачанные **без расширения**).

## Возможности

- 📄 Открытие одного файла, нескольких файлов или **целой папки** (рекурсивно).
  Расширение не важно — приложение само определяет логи по содержимому.
- 🖱 Drag & drop файлов и папок прямо в окно.
- 👀 Два режима просмотра: **«Чистый вид»** (карточки сообщений) и
  **«Исходный JSON»**.
- 📋 Копирование одной кнопкой: весь чат / только промты / только ответы /
  одно сообщение (кнопка на каждой карточке) / исходный JSON.
- 💭 Размышления модели (thoughts): показать/скрыть в просмотре; в экспорте —
  не включать / включать в сообщения / **отдельным файлом** `*_thoughts`.
  Логи без размышлений обрабатываются без проблем.
- 💾 Экспорт в **TXT / HTML / Markdown** с настройками:
  - нумерация сообщений (да/нет)
  - метки времени (да/нет)
  - шапка с моделью и параметрами (да/нет)
  - системная инструкция (да/нет)
  - вложения: плейсхолдер + ссылка на Google Drive (да/нет)
  - HTML: рендер Markdown (код-блоки, жирный, списки…) или сырой текст
  - свои подписи ролей (ПОЛЬЗОВАТЕЛЬ/МОДЕЛЬ → что угодно)
- 📦 Массовый экспорт: кнопка «Экспорт всех» обрабатывает весь список разом.
- 🌙/☀️ Тёмная и светлая темы (настройки запоминаются между запусками).
- 🛡 Отказоустойчивость: битые/чужие файлы пропускаются с отчётом,
  оборванные генерации (размышления без ответа) не роняют программу.

## Установка и запуск

```bash
pip install PySide6
python app.py
```

Можно сразу передать файлы/папки аргументами:

```bash
python app.py chat_log1 chat_log2 ./папка_с_логами
```

## CLI (без GUI, для массовой обработки)

```bash
# информация о файле
python cli.py chat_log

# экспорт папки в HTML, размышления отдельным файлом
python cli.py ./logs -f html --thoughts separate -o ./out

# TXT без нумерации, с метками времени
python cli.py chat_log -f txt --no-numbering --timestamps -o ./out
```

Все опции: `python cli.py --help`

## Структура проекта

| Файл      | Назначение                                           |
|-----------|------------------------------------------------------|
| `core.py` | Ядро: парсинг лога, модели данных, экспортёры, Markdown-рендер. Без зависимостей, чистый Python. |
| `app.py`  | GUI на PySide6                                       |
| `cli.py`  | Консольная утилита                                   |

## Формат лога AI Studio (кратко)

```jsonc
{
  "runSettings":      { "model": "...", "temperature": 1.0, ... },
  "systemInstruction": { },
  "chunkedPrompt": {
    "chunks": [
      { "role": "user",  "text": "...", "tokenCount": 117, "createTime": "..." },
      { "role": "model", "isThought": true, "text": "..." },   // размышления
      { "role": "model", "text": "...", "finishReason": "STOP" },
      { "role": "user",  "driveImage": { "id": "..." } }        // вложение
    ],
    "pendingInputs": [ ... ]
  }
}
```

Парсер склеивает подряд идущие чанки пользователя (вложение + текст),
прикрепляет размышления к следующему ответу модели и переживает любые
отсутствующие поля.
